Persetujuan

  • IP Berganti Setiap Kali Ada Notifikasi Panel Baru.
  • Harus Online Setiap Hari Di Discord Dan Pantau Announcement.
  • Harus menerima resiko jika server down karena keberatan.
  • Setiap server diberi akses backup sebanyak 40x, kamu harus memanfaatkan itu dengan sering backup.
  • Ping ada di 20-90 Di jamin aman buat server besar.
  • Kita sangat menghindari ddos, jadi kita akan mengganti IP semua server di layanan kamu setiap update panel.
  • Harus menyembunyikan password akun agar tidak ada orang lain yang login ke akunmu.
